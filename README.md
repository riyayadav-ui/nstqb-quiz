# NSTQB Quiz – Vercel Deployment Guide

## 🚀 Deploy to Vercel in 5 Minutes (Free)

### Step 1 — Create a GitHub account
Go to https://github.com and sign up (free)

### Step 2 — Create a new GitHub repository
1. Click **+** → **New repository**
2. Name it: `nstqb-quiz`
3. Set to **Public**
4. Click **Create repository**

### Step 3 — Upload project files
1. Click **uploading an existing file** on the repo page
2. Drag & drop ALL files from this zip (keep folder structure)
3. Click **Commit changes**

### Step 4 — Deploy on Vercel
1. Go to https://vercel.com → Sign up with GitHub (free)
2. Click **+ New Project**
3. Select your `nstqb-quiz` repository
4. Click **Deploy** (Vercel auto-detects React)
5. Wait ~2 minutes ✅

### Step 5 — Share your link!
Vercel gives you a URL like:
👉 `https://nstqb-quiz.vercel.app`

Share this with anyone — works on all devices!
